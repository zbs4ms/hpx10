## 当前目录结构

> 顺序依照编辑器显示顺序（文件夹后面一定要加`/`）

```
|—— assets/         # 全局资源（js, css, images...）
|
|—— components/     # 组件
|
|—— consts/         # 常量
|
|—— mixin/          # mixin
|
|—— plugins/        # vue插件
|
|—— router/         # 全局路由
|
|—— store/          # 全局store
|
|—— utils/          # 工具类（譬如：工具函数）
|
|—— App.vue         # 项目根组件
|
|—— main.js         # 项目入口
|
|—— rootApi.js      # 全局api
```
